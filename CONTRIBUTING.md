# Contributing / Mitwirken

🇬🇧 [English](#english) | 🇩🇪 [Deutsch](#deutsch)

---

## English

### How to Contribute
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Run linting: `ruff check custom_components/smart_garage/`
4. Update translations (both `de.json` and `en.json`)
5. Update `CHANGELOG.md`
6. Submit a Pull Request

### Development Setup
```bash
git clone https://github.com/thokoh74-DE/smart-garage.git
cd smart-garage
pip install ruff
```
