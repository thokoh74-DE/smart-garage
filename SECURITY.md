# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |

## Reporting / Melden

🇬🇧 Report vulnerabilities via [GitHub Security Advisories](https://github.com/thokoh74-DE/smart-garage/security/advisories/new). Do **not** open public issues.

🇩🇪 Schwachstellen bitte über [GitHub Security Advisories](https://github.com/thokoh74-DE/smart-garage/security/advisories/new) melden. **Kein** öffentliches Issue erstellen.
